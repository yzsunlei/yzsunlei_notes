## 水平垂直居中实现方式

### 居中元素定宽高

- absolute + 负margin
- absolute + margin auto
- absolute + calc

### 居中元素不定宽高

- absolute + transform
- lineheight
- writing-mode
- table
- css-table
- flex
- grid
